#  Sistema CRUD de Productos

Sistema de gestión de inventario por línea de comandos construido en Python.

## Características

- Consultar precios de productos
- Agregar nuevos productos
- Actualizar precios existentes
- Eliminar productos
- Ver catálogo completo con estadísticas

##  Cómo usar
```bash
python3 main.py
```

##  Tecnologías

- Python 3.x
- Diccionarios
- Validación de entrada

## Aprendizajes

Proyecto desarrollado para practicar:
- Operaciones CRUD
- Estructuras de datos en Python
- Validación de entrada de usuario
- Menús interactivos
